Rounds

  • Objective Round
    1. Computer Science
    2. Quants
    3. English Comprehension
    4. Logical Ability
  • Subjective Round
    1. Automata Coding ie. : Test Coding ability in C, C++, Java, Python, Ruby, C#
  • Advanced Coding Round
    1. Advanced Coding for Higher Package :- This round is for students looking for higher package placement, it will also cover the same topics as in Automata Coding

Note: Here Automata Coding is NOT Automata Fix, some students may get confused with respect to that, please keep in mind those two are different. The Persistent Systems Placement Papers will consist of approximately 2 questions to be answered in 40 minutes (excluding Advanced Coding Round).
